Algorithm Practice is an interactive web application for practicing small JavaScript algorithms. Users can dynamically select different algorithms from the sidebar and enter inputs to see instant results.

Features

Dynamic Sidebar Navigation: Each sidebar option loads its corresponding input fields.

Algorithms Included:

Polygon Angle Sum

Basketball Points Calculation

Age to Days Conversion

Remainder Calculation

Cube of a Number

Less than 100 Checker

Count True Values

Min & Max from Array

Dynamic Input Fields: Single, double, or array inputs change based on the selected algorithm.

Result Display: Shows calculated result immediately.

History Tracking: Keeps track of all calculated results with timestamps.

Reset Functionality: Clear inputs, results, and history.
